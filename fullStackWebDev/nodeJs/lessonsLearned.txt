Node JS is usefull because it allows for both the backend and frontend to be written in the same language.
Async allows for multiples process to be run at the cost of system memory.
Node is good for REST APIs & microservices, CRUD apps.
Not good with CPU intensive calculations (e.g. math)

steps to create new TS env
npm init (follow instructions)
npx tsconfig.json (ben awad usefull config) // make sure code is enclosed in src/
npm run watch (tsc -w)
npm run dev ( nodemon dist/index.js)
